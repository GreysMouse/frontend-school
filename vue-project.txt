Приложение по управлению заметками.

Развернуть на Vue 3 в стиле Composition API (опционально Options API).
Дизайн: https://www.figma.com/design/YuGxLRvd4ll72xl8f96wbu/%D0%A8%D0%BA%D0%BE%D0%BB%D0%B0_frontend_%D1%80%D0%B0%D0%B7%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D1%87%D0%B8%D0%BA%D0%B0?node-id=23-854&t=X3RGmo6RLgGaXFtW-0

Элементы страницы:

1. Хэдер: название "Notes" приложения, строка поиска заметок, кнопка-ссылка перехода к архиву удаленных заметок.
2. Футер: Общее кол-во текущих заметок, подпись автора заметок.
3. Основа: кнопки переключения списка (1 - в виде плиток, 2 - в виде списка по одному на строку), карточки заметок в двух вариантах представления.
4. Карточка заметки, которая содержит:
	- порядковый номер в списке,
	- заголовок (не может быть пустым, форма будет не давать оставить его пустым)
	- текст (содержание) - может быть пустым
	- изображение (выводится крупно поверх экрана при клике на какую нибудь кнопочку или иконку на заметке, возможно сделать как thumbnail - на усмотрение дизайнера)
	- вывод даты создания (наверное надо чтобы не сильно мешало, возможно мелким шрифтом)
	- кнопка (три точки) для открытия меню взаимодействия (дропдаун), где есть кнопки редактировать и удалить заметку (возможно просто иконками - реализация на усмотрение дизайнера)

5. Кнопка добавления новой заметки (хз куда, можно просто поверх футера и карточек в виде крестика в круге). При нажатии на которую открывается модалка по середине экрана (крупная), содержащая:
	- заголовок типа "создание заметки" или другой
	- поле ввода заголовка (со звездочкой, поскольку обязательно) 
	- поле ввода текста
	- кнопка прикрепления изображения
	- кнопка подтвержения создания
	- кнопка отмены создания

Пагинация - бесконечный скролл вниз.
Пожелания - не сильно сложную верстку, лучше чтобы было переиспользование компонентов и какая нибудь дизайн-система, иначе не успеют. Лучше в заметки в фигме вынести комменты типа "при клике мимо карточки менюшка закрывается", про ховеры и т. д.


Обязательные задания для практиканта:

1. Верстка списка элементов (карточек).
2. Верстка хэдера и футера
3. Верстка модальных окон и работа с формами на Vue
4. Работа с HTML-формами
5. Работа с запросами (fetch)
6. Реализация пагинации
7. Реализация поиска по элементам (карточкам)
8. Создание и устранение уязвимостей
9. Опционально: роутинг в SPA
